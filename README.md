# 全方位生活與健康追蹤儀表板

全方位生活與健康追蹤儀表板是一個以個人日常健康管理為核心的產品，將體位數據、睡眠紀錄、每日活動與視覺化分析整合在同一個介面中，幫助使用者持續記錄、回顧與理解自己的生活狀態。

## 產品願景

多數健康資料分散在體脂計、手寫筆記、手機備忘錄或穿戴裝置中，難以形成可以長期觀察的個人資料庫。這個產品的目的，是把每天真正重要的健康訊號集中起來，讓使用者用更低成本的方式看到自己的變化趨勢，並逐步建立對生活習慣與身體狀態之間關聯的理解。

## 解決的問題

- 體重、體脂、肌肉率等數據分散，缺乏統一紀錄入口
- 睡眠品質與半夜異常醒來等訊號不容易長期追蹤
- 步數與通勤型態等生活因子難以和身體數據一起觀察
- 缺少一個屬於個人的健康儀表板，用來看趨勢而不是只看單次數值

## 核心體驗

### 體位數據追蹤

使用者可以記錄體重、體脂率、肌肉率與內臟脂肪，並以時間軸檢視變化。系統支援日期區間查詢，也能處理同一天多筆紀錄的顯示策略。

### 睡眠與異常喚醒觀察

睡眠功能不只記錄入睡與起床時間，也會自動計算睡眠時長，並標記凌晨 3 點到 4 點之間的異常醒來。這些事件會在後續的趨勢視覺化中成為可以被觀察的訊號。

### 每日活動與通勤因子

除了身體數據，產品也記錄每天步數、通勤方式與通勤時長，讓使用者能把活動量與體位變化放在同一個脈絡裡理解。

### 儀表板視覺化

儀表板會提供體重、體脂率、肌肉率的趨勢圖，並把睡眠異常與步數熱度一併疊加，讓資料不只是被儲存，而是能被閱讀。

## 主要功能

- Google OAuth 2.0 登入
- JWT 驗證與 Refresh Token 機制
- 體位數據 CRUD 與時間區間查詢
- 睡眠紀錄 CRUD 與異常喚醒自動標記
- 每日活動與通勤紀錄
- 以使用者為單位的資料隔離
- 健康儀表板與趨勢圖分析
- PWA 離線使用體驗

## 產品特色

- 以個人健康追蹤為中心，而不是通用筆記工具
- 同時整合生理數據與生活因子
- 預設以 Asia/Taipei 時區理解日常生活脈絡
- 使用圖表與事件標記協助發現關聯
- 以 Google OAuth + JWT 保護個人隱私與資料安全

## 使用情境

- 每天早上量測體重與體脂後快速輸入紀錄
- 回顧最近 30 天的睡眠品質是否影響體重波動
- 比較通勤與步數高低是否和身體組成變化相關
- 長期建立自己的健康資料庫，作為自我觀察依據
